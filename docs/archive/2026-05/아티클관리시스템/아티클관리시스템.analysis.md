### 가중 평균 = 94%

- 영역 1~7은 동일 가중치(× 1.0)
- 영역 8(검색)은 × 1.2 (전체 UX 영향)
- 영역 9(회귀)는 × 0.8 (가시화는 충분, 자동화는 후속)
- 가중 합 9.42 / 가중치 합 10.0 = **94.2%**

## 7. 동작 확인 시나리오 (수동 검증 권장)

다음 4가지 시나리오는 매니저 로그인 후 운영 환경 진입 시 직접 확인 권장:

1. **매니저 작성 흐름**: `/admin/articles/new` → content_type 선택 → 메타 입력 → 본문 작성 → 검증 패널 실시간 갱신 → draft 저장 → 발행
2. **content_type 변경 시 URL 자동 보존**: published 글 content_type 변경 → `article_redirects` UPSERT → 옛 URL 접근 시 308
3. **호텔리어 검색**: `/search?q=CI` → synonyms 확장 → "체크인" keywords 매칭 → 결과 노출. content_type 칩 필터링 동작
4. **회귀 검증**: `/admin/sandbox/markdown-matrix` → 14종 스타일 적용 → 우측 markdown(+ 인라인 HTML) 출력 시각 확인
