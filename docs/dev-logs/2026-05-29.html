🔍 운영 테스트 체크리스트 (수동)

  • 10곳 페이지 모두 RichEditor 마운트 + WYSIWYG 입력 정상
  • 이미지 업로드: 톨바 클릭 → 모달 → 파일 선택 → 본문에 삽입
  • 이미지 리사이즈: 2000px 이상 파일 업로드 → 1600px로 축소 확인
  • 링크 입력: Cmd+K → 모달 → URL → 적용 / 편집 / 제거
  • 빠른답변: Cmd+/ → 패널 → 검색 → ↑↓ → Enter / Ctrl+1~9
  • 변수 치환: {{호텔명}} 포함 템플릿 삽입 시 자동 치환 (매니저 답변 시점)
  • SMS 미리보기: 공개 답변 본문 입력 → 토글 펼침 → 카운터 + plain text
  • 자동 저장: 본문 입력 → 30초 후 "저장됨" 표시
  • draft 복구: 작성 도중 탭 닫고 재진입 → 다이얼로그 노출 → 복구 동작
  • draft 자동 삭제: 발행 후 재진입 → 다이얼로그 안 뜸
  • 호텔리어 placeholder: /tickets/new 본문 빈 상태 → "예시 채우기" 클릭 → 템플릿 prefill
  • 모바일 톨바: 모바일 viewport(< 768px) + lite 모드 + 에디터 focus → 하단 톨바 노출
  • 단축키 도움말: F1 또는 Cmd+? → 모달
  • 다크모드: 모든 모달·패널·톨바 다크 톤 가독성 확인
  • viewMode 전환: 매니저 → 호텔리어 시점 → 톨바 색상 자동 변경 (brand-* cascade)
