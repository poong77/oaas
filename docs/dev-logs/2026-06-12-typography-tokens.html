1. 무엇을 했나

디자이너가 Figma에서 export한 폰트 스타일 35종을 프로젝트의 디자인 토큰으로 흡수했습니다.

  • 원본 CSS 파일은 import하지 않음 — Tailwind 4 전용 문법 --text-{name}으로 @theme에 등록하여 text-{name} 유틸리티가 자동 생성되도록 함.
  • size + line-height + font-weight가 한 클래스에 묶여 적용됨. 예) text-heading-large-bold = 32px / 1.48 / 700.
  • 폰트 패밀리는 전역 --font-sans(Pretendard) 상속, 색은 기존 text-brand-*/text-slate-* 유지 — 타이포와 컬러 역할 분리.

3. 이름 결정 사항

display-xlarge-medium → display-xlarge-black 으로 변경.
Figma 원본 스타일명은 'Medium'이었으나 실제 weight가 900(Black)이어서, 이름·값 불일치를 없애기 위해 black으로 확정(디자인 확인 완료). 미사용 토큰이라 안전하게 개명.
