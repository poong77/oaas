인터랙션 명세

1통계 카드
전역 카운트(필터 무관). 카드 클릭 → ?status=all|active|inactive. 현재 상태는 테두리 강조.

2검색·필터
검색(이름·아이디·이메일·연락처) Enter 적용 · 권한 · 상태 · 정렬. 변경 시 1페이지로.

3결과 바
실시간 결과 수 · 로딩 표시 · 필터 활성 시 ‘초기화’.

4행 클릭
행 전체 클릭/Enter → /admin/users/[id] 편집. hover 하이라이트 + 우측 chevron.

5아바타
이름 첫 글자 + ID 해시 6색. 비활성은 흑백·반투명.

6상태
도트(활성=녹색·비활성=회색) + 라벨. 비활성 행은 배경 톤다운.

7페이지네이션
현재 범위(1–20 / N명) + 이전/다음. 1페이지면 숨김.
