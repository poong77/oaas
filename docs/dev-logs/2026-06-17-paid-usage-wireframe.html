구현 메모

  • 위치: 마스터DB ⑤ 시스템·운영 → 운영시간 · 메뉴 접근 제어 · [유료 사용현황] (맨 우측, '비용' 뱃지)
  • 접근: requireRole(['admin']) — 매니저 차단(menu-access 토글 없는 어드민 고정)
  • AI 데이터: ai_usage_logs 신규 테이블 — 배포 시점 이후 누적 / 문자: notification_logs 과거 소급
  • 탭 전환은 서버에서 3개 기간 리포트를 모두 받아 클라이언트 토글(추가 패칭 없음)
  • 막대 길이 = 버킷별 (AI비용×환율 + 문자요금) ÷ 기간 내 최대값
  • 용도별 내역: ai_usage_logs.bucket 집계 → 한글 라벨 매핑(ticket-draft→티켓 답변 초안 등). 3개 묶음(답변·콘텐츠 생성 / 추천·보조 / 검색 인프라)으로 색 구분
