🎯 기능 요약

  • 채널 검색 — 채널명(부분일치) 또는 채널ID(C…/G…) 둘 다. conversations.list / conversations.info 자동 분기
  • 봇 초대 — 공개 채널은 conversations.join으로 자동입장 → 연동 완료 첫 메시지 자동 게시
  • 로고 회색↔컬러 — 봇 참여(bot_joined=true) 채널 1개 이상이면 슬랙 로고 컬러
  • 비공개 채널 — 저장하되 봇 미초대 표시 + /invite @봇 안내 + [상태 새로고침]
  • N:N · 연동해제(soft delete) · 테스트 발송 지원
  • 접수 알림 병행 — 접수 시 bot_joined=true 채널 전체로 발송 (기존 #support_new 병행, 기본채널 중복 skip)
