동작 규칙

  • 봇이 참여(botJoined)한 채널에만 토글 노출 — 미참여 채널은 어차피 발송되지 않음
  • 정지 시 헤더 옆 알림 정지 배지 표시, 버튼 호박색 강조
  • 테스트 발송은 정지 상태와 무관하게 동작 (연결 확인용)
  • 연동해제 → 재연동 시 notify_enabled는 true로 복귀

점검 결과 통과

  • tsc --noEmit 전체 통과 — 타입 오류 없음
  • 액션명/필드명 충돌 없음 (notify_toggle, notifyEnabled 신규)
  • listHotelSlackChannels 사용처(호텔 상세 page.tsx) 정합 확인
  • 엣지 케이스(정지→해제→재연동) 보강 완료
  • 이번 기능 6개 파일만 선별 커밋 (legacy import 등 타 WIP 제외)

DB 적용

.env.local이 운영 DB(oaas_prd, Tailscale 5433)를 직접 가리킴. db:migrate 저널 손상(0038 누락)으로 인해 ALTER TABLE ... ADD COLUMN IF NOT EXISTS 직접 DDL로 적용. 적용 후 information_schema 조회로 컬럼 존재 검증 완료 → main 푸시 자동 배포와 스키마 정합성 일치.
