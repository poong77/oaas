9. 빌드/검증 안내

사용자가 실행해야 할 명령

  1. npm install — 신규 의존성 (next-auth, bcryptjs, @aws-sdk/client-sesv2, zod, tsx 등)
  2. npx drizzle-kit generate — 마이그레이션 SQL 생성 (db/migrations/0000_*.sql)
  3. npm run build — 프로덕션 빌드 (DB 없이도 통과)
  4. npm run lint — 린트 (warning만 허용)

실 DB 연결 후 (별도 단계)

  1. 통합AS 전용 Neon URL 받기 → .env DATABASE_URL 교체
  2. npm run db:push — 스키마 반영 (또는 generate된 SQL을 migrate)
  3. npm run db:seed — 시드 데이터 입력
  4. npm run dev 후 http://localhost:3000/login에서 admin@oa.local / oa1234! 로그인 테스트
