CB-05 — 지식팩 내보내기 (AI 지식 가공)

발행 아티클·FAQ·동의어를 GPT-4o mini 최적 포맷(Markdown / JSONL)으로 가공해 다운로드

2026-06-02 Phase A·B·C 완료 빌드 ✓ / 타입체크 ✓ 매니저·어드민

요약

챗봇(GPT-4o mini)에 공급할 지식을 한 곳에서 내려받는 도구를 추가했습니다. 사용자 요청대로 Markdown / JSONL 2종만 생성하며, AI 인식률·토큰 효율이 낮은 PDF는 제외했습니다. 본문의 인라인 HTML을 정리하고, 동의어를 "용어 사전"으로 문서 상단에 인라인하며, AI 사용 지침 프리앰블을 함께 담아 날것의 데이터가 아닌 "고품질 답변에 최적화된" 형태로 가공합니다.

발행본 로드
published · is_active → 본문 정규화
HTML 노이즈 제거 → 동의어 용어사전
구어체→표준어 → self-contained 청크
제품·카테고리 부착 → MD / JSONL
  • 본문 정규화: HTML 주석 제거 · <img> → [이미지: alt] · <br>/문단태그 → 개행 · 스타일 전용 태그(span/div/font) 언랩 · 잔여 태그 제거 · 과다 개행 축소
  • 용어 사전: 활성 동의어 그룹을 카테고리별로 표준어 ← 이형어1, 이형어2 형태로 인라인 → 모델이 구어체 질문을 표준어에 매핑
  • AI 사용 지침: "지식에 있는 내용만 근거로 답하라 · 못 찾으면 이슈 접수 안내 · 출처 [slug] 표기"를 문서 최상단에 고정
  • 출처 URL: /help/{product}/{content_type}/{slug} — 챗봇 인용 링크
