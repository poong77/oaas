인터랙션 명세

1통계 카드
전역 카운트. 클릭 → ?status= / 연락처없음 필터. 현재 상태 테두리 강조.

2검색·필터
호텔명(띄어쓰기·하이픈 무시) Enter 적용 + 상태 ▾. 변경 시 1페이지로.

3결과 바
결과 수 + 필터 시 ‘초기화’.

4행 클릭
행 전체 클릭/Enter → 편집 폼. hover 하이라이트 + chevron.

5아이콘 작업
✎ 편집 · ⏻ 활성/비활성 토글(비활성 시 빨강). 텍스트 라벨 제거, hover 시 툴팁.

6상태 도트
녹색=활성 · 회색=비활성. 비활성 행 톤다운.

7＋ 추가
인라인 폼 토글. 아이콘 라벨 입력.

핵심 변경:
① 작업 텍스트버튼(“수정·비활성”) → 아이콘(✎ ⏻)
② 상태 텍스트뱃지 → 컬러 도트
