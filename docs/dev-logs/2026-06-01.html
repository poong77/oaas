검증 결과

  • 공지 66건 전부 INSERT 성공 (DB 총 71건 = 신규 66 + 기존 seed 5)
  • 이미지 58건 Vercel Blob 업로드 → 본문에 ![](blob URL) 인라인 임베드
  • Blob 이미지 공개 접근 HTTP 200 확인 (원본과 동일 바이트수)
  • 작성일을 published_at·created_at에 KST→UTC 보존, 전부 발행 상태
  • author_id = OA 어드민 계정 매핑 확인
  • HTML→Markdown 변환(줄바꿈·링크·이미지·엔티티 디코딩) 정상
  • incident 14건 = 실제 서비스 장애만 / release 6건 = 실제 업데이트만 (오분류 0)

재실행 방법

# 미리보기 (DB/Blob 미변경, JSON·HTML 리포트만 생성)
npx tsx scripts/migrate-as-notices.ts

# 실제 적재 (Blob 업로드 + notices INSERT, 중복 자동 skip)
npx tsx scripts/migrate-as-notices.ts --commit
